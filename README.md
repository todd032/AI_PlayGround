# AI_PlayGround

캐주얼 웹게임 모음 저장소입니다. 각 게임은 `games/<slug>/` 폴더에서 독립 실행됩니다.

## 실행
```bash
python3 -m http.server 4173
```
- 런처: `http://localhost:4173/`

## 게임 목록
- Meteor Dodge
- Mini Minesweeper
- Tic-Tac-Toe
- 2048 Lite
- Snake
- Brick Breaker
- Flappy Lite
- Memory Match
- Tetris Lite
- Color Reflex
